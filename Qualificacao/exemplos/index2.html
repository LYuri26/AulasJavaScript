Introdução ao CSS

A base para trabalhar estilos e aparência nas páginas web.

O que é CSS?

CSS é a linguagem que define cores, tamanhos, espaçamentos, fontes, layouts e animações, separando forma e conteúdo.

Seletores CSS

A forma como você seleciona e estiliza qualquer elemento.

Seletores Simples

  • h1, h2, p, nav (por tag)
  • .classe (por classe)
  • #id (por ID)
  • [atributo] (por atributo)

Seletores Combinados

  1. pai filho → div p
  2. pai > filho direto → div > p
  3. irmão próximo → h1 + p
  4. todos os irmãos → h1 ~ p

Exemplo de Caixa
